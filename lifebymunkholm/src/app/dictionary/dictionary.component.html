Dictionary for programming

CSS
Cascading Stylesheet - used for styling and designing a webpage.
HTML
Hypertext Markup Language - is concerned with the content and structure of the website.

HTML

Tags
Tags are angle brackets with a single word enclosed inside, where each tag has a specific meaning. Most tags in HTML have a correspnoding end tag.
<img>
Used to add images to a page. Example: <img src="someimg.jpg" height="100px" width="100px" alt="just some image">
<meta>
The meta tag is used to provide keywords and descriptions for search engines. Example: <meta name="keywords" content="HTML, CSS"> or <meta name="description" content="This is my first website.">
<p>
Paragraph tag is used to add text to a page.
<a>
Is used to insert a hyperlink. Example: <a href="http://www.google.com">Click here to go to Google </a>
<ol> and <li>
Is an ordered list and is used to create a list with numbers or alphabets as list markers. Example:
<ol>
<li>Chocolate</li>
</ol>
<ul> and <li>
Is an unordered list that use other shapes a dots or hollow circle.
<table>, <tr>, <th>, <td>
Is used to create tables, which is created row by row. <tr>="Table row", <th>="Table Header", <td>="Table Data",.
<div>
Division is used to define a division or a section. It is normally used together with CSS to format the content.
<span>
Span is similar to div, but the main difference is taht div is a block element, while span is an inline element. Is used to wrap small portions of text, images etc.
Block element
Is one that starts and ends with a new line break.
Inline element
Is one that does not starts and ends with a inline break.
<script>
The script tag is used to add internal Javascript code to our HTML document or to link to an external script.
<strong>
Make the text bold.
<em>
Displayes the text in italics.
