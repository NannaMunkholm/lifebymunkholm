Templating - Structure your site

Setup the main folders

1. Create a new page/folder write "ng g c name of folder"
2. Add the new page to app.routing.modules.ts
3. Insert:

        
            import DictionaryComponent from './dictionary/dictionary.component'
            path: 'dictionary', component: DictionaryComponent

Create a new page within the folder, insert in the terminal:
"ng g c name of folder/name of new page"
