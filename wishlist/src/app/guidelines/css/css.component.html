Main struture of a page

A HTML document is made up of two main elements, the head and the body elements.
The head provides general information about the document, incl. its meta data, title and links to additional resources.
Within the head tags, other tags that provide behind-the-scene information is enclosed.
All content provided within the body tags will be displayed on the webpage. 

          
              <head>
              <meta charset="UTF-8">
              <meta name="description" content="Free Web tutorials">
              <meta name="keywords" content="HTML,CSS,XML,JavaScript">
              <meta name="author" content="Nanna Bach Munkholm">
              <meta name="viewport" content="width=device-width, initial-scale=1.0">
              </head>
      
              <body>
              </body>

Within the body tags we first have the header.
The header defines the top section of a webpage and normally consist of a logo/banner of the website.
Afterwards the navigation section is given with <nav>
The main section is afterwards enclosed by <main>
Last the footer element is given to define the bottom of the webpage.
